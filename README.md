## Go Clutch Scraper

Scrape [Clutch](https://clutch.co/) with [Go](https://go.dev/).


### Getting Started

Run the commands below in your terminal: 

```sh
go mod init clutch
go mod tidy
go build
./clutch

## Extracted data is inside the data directory.
```
